# Chethana R V - Personal Academic Website

Personal academic website of Chethana R V, PhD researcher at Dayananda Sagar University, Bengaluru.

## About

This website showcases my research in Quantum Cryptography, Post-Quantum Cryptography, and related areas. Built using the [al-folio](https://github.com/alshedivat/al-folio) Jekyll theme.

## Research Focus

- Quantum Computing & Cryptography
- Post-Quantum Cryptography
- Kolmogorov Complexity
- One-way Functions
- Cyber Security

## Local Development

To run this site locally:

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000` in your browser.

## Deployment

This site is deployed automatically to GitHub Pages when changes are pushed to the main branch.
